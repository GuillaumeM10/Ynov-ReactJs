# API site

https://www.omdbapi.com/

# How to run

1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000/` in your browser

## Features

- [] User
  - [] Signup
  - [] Login
  - [] Logout
  - [] Profile
  - [] Profile Edit
  - [] Change Password
  - [] Likes
  - [] Comments
  - [] role
